# MyHome

Панель климат-контроля спальни с интеграцией Яндекс Smart Home. Показывает температуру, влажность, погоду на улице, оценку сна и рекомендации. Присылает уведомления если жарко, сухо или кондиционер выключен.

## Настройка

Скопируйте `.env.example` в `.env` и заполните:

```env
YANDEX_OAUTH_TOKEN=токен
ROOM_FILTER=спальня
WEATHER_CITY=Севастополь
WEATHER_LAT=44.9521
WEATHER_LON=34.1024
```

## Запуск

### macOS-приложение

```bash
./scripts/build-macos-app.command
open ~/Desktop/MyHome.app
```

Приложение запускает локальный сервер в фоне, открывает окно панели и раз в минуту проверяет критические показатели — присылает уведомления если температура >22°C, влажность вне нормы или кондиционер выключен.

### Ручной запуск сервера

```bash
./scripts/start.command
# или
node server/index.ts
```

Панель будет доступна на http://localhost:4173.

## Vercel

Деплой одной командой:

```bash
npm i -g vercel
vercel
```

Переменные окружения (`YANDEX_OAUTH_TOKEN` и др.) нужно добавить в Vercel Dashboard.

## API

| Endpoint | Описание |
|---|---|
| `/api/iot` | Датчики Яндекс Smart Home (отфильтрованные по `ROOM_FILTER`) |
| `/api/weather` | Погода с Open-Meteo (кеш 1 час) |
| `/api/alerts` | Критические показатели: температура, влажность, заряд батареи, статус кондиционера |
| `/api/config` | Сетевые URL сервера |

## Главная панель

На главном экране — климатическая аналитика спальни:

- **Температура** с оценкой комфорта и счётом сна
- **Влажность** с меткой (сухо / норма / влажно)
- **Погода на улице** — температура, влажность, ветер
- **Кондиционер** — состояние, целевая температура, режим
- **Рекомендации** — учитывают работу кондиционера, срочные выделены красным
- **Датчики** — сетка карточек всех устройств в комнате

Значения приоритетно берутся с датчика климата (тип `sensor.climate`), игнорируя температуру кондиционеров и других приборов.

## Стек

| Слой | Технология |
|---|---|
| **Фронтенд** | React 19 + TypeScript + Vite |
| **Сервер** | Node.js (ESM, без фреймворков) |
| **macOS** | Swift (Cocoa + WKWebView), нативные уведомления |
| **Погода** | Open-Meteo (бесплатно, без API-ключа) |
| **Деплой** | Vercel (serverless API + CDN статика) |

## Структура проекта

```
src/          фронтенд (React, Vite)
  components/  ClimateHero, WeatherPanel, Dashboard, SensorCard
  hooks/       useIot, useWeather
  lib/         climate, sensors, api
  types/       TypeScript-типы
server/       Node.js HTTP-сервер (API + раздача статики)
api/          Vercel serverless handler
macos/        Нативное macOS-приложение (Swift)
scripts/      Сборка и автозапуск
```
